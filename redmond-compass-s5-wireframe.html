Annotations

Numbered to the screen. Tier tags show what renders when.

  1. Hero / gallery STATE

    Free shows one photo; Member shows a gallery (dots appear when toggled). One hero minimum so a free listing is never bare.

  2. Name + status line ALL STATE

    The most-glanced element. Open/closed + closing time leads; states cover closed (shows next open), temporarily/permanently closed. Reviews slot reserved for v2 so adding it isn’t a redesign.

  3. Action bar — sticky ACTION

    Call · Directions · Save · Follow, pinned so it’s always one tap away as you scroll. Same four verbs as the result card (S4). Logged-out tap on Save/Follow fires the just-in-time login — the only place we ask.

  4. Module buttons MODULE

    Book (B7), Request a quote (B8), Join rewards (B9) appear only when that module is attached. This is the clearest place residents feel the membership — as added utility, never as a paywall.

  5. At a glance ALL

    Hours (today highlighted), address→map, phone, website, amenity tags. Tags use the same vocabulary as the S4 search facets — profile and filters speak the same language. Member exposes the fuller tag set.

  6. What’s new — bulletins ALL

    The follow payoff and engagement hook. Free posts up to the monthly cap; Member is unlimited + scheduled — so a member simply has more here, not a different screen.

  7. Upcoming — events ALL

    Events tied to this business. Free and uncapped for everyone — pure consumer value, never gated.

  8. About / story STATE MEMBER+

    Free shows a basic description (still complete). Member swaps in the storytelling block + owner spotlight — the local-character content that makes the directory feel like a community.

  9. Follower perks MEMBER+

    A standing offer for people who chose to follow — earned reach, the inverse of an ad. Renders nothing on a free profile (no empty stub).

  10. Bottom tabs ALL

    Persist on this pushed screen. Saved carries the Rewards wallet (the Loyalty crossover) once the resident joins a program.
