S5 Reputation

  1. Trust signals

    Factual, automatic badges — verified, posts weekly, replies-in-a-day, local-since. No opinion, just whether the business shows up and responds. These render for every tier, so even a free listing earns real trust.

  2. Recommendations — positive-only

    A count that can only rise, so it can’t be review-bombed. Notes are short and come from people who actually transacted (Verified customer badge, drawn from booking/loyalty). The sub-line states plainly: no 1–5 score.

  3. Recommend action

    One tap (the demo). Logged-out fires the just-in-time login; an optional short note can be added. It only ever adds a positive — there is no negative direction to weaponize.

  4. Private feedback routing

    The anti-Yelp move: an unhappy customer is steered to tell the owner privately, not post a public 1-star. For a Pro business this lands in the managed Inquiry inbox (service recovery); otherwise it reaches their contact. Criticism becomes a fix, not a wound.

  5. New-business state

    Toggle it: a new or quiet listing reads “New to Compass” — neutral and inviting — instead of a damning empty score. Fairness for the business, honesty for the resident.

S7 Rewards wallet

  1. Lives in Saved, as a tab

    Rewards joins Businesses / Following / Events. It appears only once a resident has joined at least one program — so it never clutters Saved for people who don’t use loyalty. Saved was already the “my stuff” return surface; rewards belong with it.

  2. Program cards

    Each business’s program with live progress — punch cards as filled dots, points as a bar. Tapping a card opens that business’s profile (S5). Progress also shows on S5 itself and can nudge from Home.

  3. Reward ready → redeem

    When earned, the card flips to a clear Redeem state. Redemption is confirmed at the counter (staff scan / owner tap) — defined back in the Loyalty setup (B9).

  4. Empty state pulls toward discovery

    Before any program is joined, the tab invites action — “find local businesses” routes into Search. An empty wallet still does re-engagement work.
